# cpdown

> 📄 Копируй любую страницу или субтитры YouTube в чистый Markdown.

**cpdown** — лёгкое расширение для браузера, чтобы сохранять полезный контент без лишнего шума: статьи, выделенный текст и транскрипты YouTube.

---

## ✨ Возможности

- 🧭 **Кнопка на панели** — нажми на иконку расширения или `Ctrl+Shift+T`
- 🎬 **Субтитры YouTube** — скопируй транскрипт текущего видео в Markdown
- 🖱️ **Контекстное меню** — кликни правой кнопкой по ссылке YouTube → **Copy subtitles**
- 🧼 **Readability / Defuddle** — извлеки чистый текст статьи со страницы
- 📋 **Копирование в буфер** — Markdown сразу готов к вставке
- 💾 **Скачать как `.md`** — сохрани результат в файл
- 🔢 **Счётчик токенов** — в тосте транскрипта отображается примерное количество токенов
- ✅ **Единый дизайн тостов** — сообщения с кнопки и из контекстного меню выглядят одинаково

---

## 🚀 Использование

### Скопировать текущую страницу

1. Открой любую страницу
2. Нажми иконку **cpdown** на панели или `Ctrl+Shift+T`
3. Используй кнопки в тосте:
   - **Copy** — скопировать Markdown в буфер обмена
   - **Save .md** — сохранить Markdown в файл

### Скопировать субтитры с текущего видео YouTube

1. Открой видео на YouTube
2. Нажми иконку **cpdown** на панели
3. Расширение извлекает транскрипт через скрытую вкладку — это стабильнее SPA-данных
4. В правом верхнем углу появляется тост с количеством токенов и кнопками **Copy** / **Save .md**

### Скопировать субтитры со ссылки YouTube

1. Кликни правой кнопкой по любой ссылке YouTube (`youtube.com` или `youtu.be`)
2. Выбери **Copy subtitles**
3. Расширение открывает видео в скрытой вкладке, извлекает транскрипт, закрывает вкладку и показывает результат на исходной странице

## 🔐 Разрешения

- `activeTab` — доступ к текущей вкладке
- `clipboardWrite` — копирование Markdown в буфер обмена
- `contextMenus` — контекстное меню для ссылок YouTube
- `scripting` — внедрение скриптов извлечения
- `storage` — хранение настроек пользователя
- `<all_urls>` — работа на любых страницах
